Ejemplos de position con Tailwind CSS

1. position: static (por defecto)

static
(flujo normal)

2. position: relative: no tiene que ver con su padre o el viewport, su posicionamiento es relativo a su posición original en el flujo normal del documento.

relative
(desplazado)

3. position: absolute, El elemento se posiciona respecto a su ancestro posicionado más cercano que tenga una posición distinta a static

soy el padre RELATIVE
absolute
(respecto al contenedor relativo)

4. position: fixed, El elemento se posiciona relativo a la ventana del navegador (viewport).

Haz scroll para ver que el cuadro rojo permanece fijo en la esquina inferior derecha.

fixed
(fijo en viewport)

5. position: sticky

sticky
(se queda pegado arriba al hacer scroll)

Contenido para hacer scroll dentro del contenedor amarillo.
